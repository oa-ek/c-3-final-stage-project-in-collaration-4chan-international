ОФІЦІЙНИЙ ЗВІТ
Про розробку програмного продукту «Soulslike Build Simulator»

1. Вступ та мета проєкту
Метою проєкту стала розробка спеціалізованого програмного інструменту — симулятора та калькулятора характеристик персонажів для серії ігор Dark Souls та Elden Ring.

Головне завдання продукту полягало у вирішенні проблеми жорсткого обмеження ігрових ресурсів: програма повинна надавати користувачам можливість безперешкодно проєктувати конфігурації (білди) персонажів, тестувати комбінації спорядження та отримувати точні прогнози підсумкових ігрових показників без ризику втрати прогресу.

2. Реалізований функціонал (Що було зроблено)
В рамках виконання проєкту було спроєктовано та реалізовано наступні програмні модулі:

Модуль конструювання персонажа (Character Builder): Розроблено інтерактивний інтерфейс для покрокового введення базових характеристик (Strength, Intelligence тощо) та вибору екіпірування (зброя, броня, аксесуари).

Автоматизований калькулятор показників (Real-time Analytics): Впроваджено систему миттєвого розрахунку ключових ігрових метрик, зокрема:

Потенційної вихідної шкоди (Damage Output).

Комплексних показників захисту (Defense & Resistances) від фізичних та магічних впливів.

Рівня навантаження (Equipment Load) для автоматичного моніторингу ваги та запобігання штрафам на мобільність персонажа.

Математична модель масштабування (Scaling System): Імплементовано складні алгоритми обчислення приросту шкоди на основі рангів скейлінгу зброї (від S до D) залежно від поточних характеристик героя.

Управління даними (CRUD-операції): Створено надійну систему для створення, читання, оновлення та видалення ігрових предметів і збережених конфігурацій, що дозволяє користувачам зберігати свої напрацювання в базі даних.

Адаптивний дизайн: Інтерфейс програми динамічно налаштовується під специфічну естетику та набори зброї кожної з чотирьох підтримуваних ігор.

3. Використані технології та архітектура
Проєкт реалізовано з використанням сучасної клієнт-серверної архітектури, що забезпечує високу продуктивність обчислень та інтерактивність користувацького інтерфейсу. Основний програмний стек включає:

Серверна частина (Backend): * C# / ASP.NET Core — використано для побудови надійної архітектури серверної логіки, обробки CRUD-операцій, взаємодії з базою даних та виконання складних математичних розрахунків ігрових показників.

Клієнтська частина (Frontend): * TypeScript (складає близько 68% кодової бази) — забезпечує сувору типізацію та стабільну роботу інтерактивних елементів інтерфейсу у режимі реального часу.

CSS / Сучасні UI-фреймворки — застосовано для стилізації застосунку та забезпечення тематичного візуального оформлення.

База даних: * Використано реляційну СУБД (на базі SQL) для структурованого зберігання статичних довідників предметів та користувацьких профілів зі збереженими конфігураціями.

4. Висновок
У результаті виконання проєкту створено повноцінний, масштабований програмний продукт. Застосований стек технологій (TypeScript та C# .NET) дозволив успішно реалізувати всі заплановані функції, зокрема інтеграцію складних математичних моделей гри та автоматизований розрахунок параметрів екіпірування. Продукт готовий до використання цільовою аудиторією для ефективного планування ігрового процесу.
